# Improving the Reliability of Climate-Resilient Cover Cropping Systems in the Maldives through AI-Enabled Monitoring and Decision Support

## Project Overview
This project applies Advanced AI techniques (Clustering, Classification, Regression, and Neural Networks) to improve agricultural reliability in the Maldives using NDVI and Rainfall data.

## Module Details
- **Module:** Advanced Artificial Intelligence (UFCFUR-15-3)
- **Institution:** Villa College / UWE Bristol
